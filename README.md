# Calculadora de IVA

Calculadora simple para obtener el desglose de Neto, IVA y Total usando la tasa de IVA de Chile por defecto: 19%.

## Uso

Abre `index.html` en el navegador.

Puedes calcular desde:

- Monto neto
- Monto total con IVA incluido

La calculadora permite ajustar la tasa de IVA, copiar el desglose e imprimir o guardar el resultado como PDF.

## Archivos

- `index.html`: aplicacion completa en HTML, CSS y JavaScript.
